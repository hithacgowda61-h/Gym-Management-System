# 🏋️ Gym Management System

## 📌 Description
The Gym Management System is a Java-based desktop application designed to manage gym operations efficiently. It helps in handling member registration, attendance tracking, trainer management, and billing processes through a simple and user-friendly interface.

---

## 🚀 Features
- Member Registration and Login
- Trainer Management
- Attendance Tracking System
- Billing and Payment Management
- Dashboard for quick overview
- Secure password handling

---

## 🛠️ Technologies Used
- Java (Swing for UI)
- JDBC (Database Connectivity)
- MySQL (Database - if used)

---

## 📂 Project Structure
- `Main.java` – Entry point of the application  
- `LoginPage.java` – Handles login functionality  
- `RegisterPage.java` – New member registration  
- `Dashboard.java` – Main interface/dashboard  
- `MembersModule.java` – Member management  
- `TrainersModule.java` – Trainer management  
- `AttendanceModule.java` – Attendance tracking  
- `BillingModule.java` – Payment and billing  
- `DB.java` – Database connection  
- `PasswordUtils.java` – Security and password handling  
- `ThemeManager.java` – UI styling  

---

## ▶️ How to Run
1. Open the project in Eclipse IDE  
2. Configure database connection (if applicable)  
3. Run `Main.java` file  
4. Use the application interface
